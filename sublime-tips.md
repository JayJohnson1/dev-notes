#Useful Sublime Tips

### ST3 Packages to install:
* Package Control
* Emmet
* Sidebar Enhancements
* Bootstrap 3 Snippets
* MarkdownEditing
* Markdown Preview

### Goto Anything in ST3
```
command + p
```

### Show/Hide Sidebar
```
command + k followed by command + b
```

### Command Palette
```
shift + command + p
```

### Install a Package
```
shift + command + p
packin
```

### Uninstall a Package
```
shift + command + p
packr
```
