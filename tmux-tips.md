#Useful TMUX Tips

### Start up TMUX
```
$ tmux
```

### Show clock
```
$ tmux clock
```

### Split pane vertically
```
$ control-b, % -or- $ control-a, \
```

### Split pane horizontally
```
$ control-b, â€œ -or- $ control-a, -
```

### Move to different panes
```
$ control-b, (arrow key) -or- $ control-a, (arrow key)
```

### Kill current pane
```
$ control-b, x (yes)
```
